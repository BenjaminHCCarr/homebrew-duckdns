DuckDNS

automatically update duckdns via launchd on OSX

Download this project as a .zip file Download this project as a tar.gz file

RIP in piece DynDNS

DynDNS is no longer free. Don't worry though, DuckDNS offers the exact same service. Now if only there was a simply way to get OSX to update my IP address with my DuckDNS host name. Oh wait -- there is!

installation

  1. $ brew install duckdns
  2. Make a $HOME/.duckdns config file like so:
DOMAIN="example"
TOKEN="11111111-1111-1111-1111-111111111111"
